01 · Fondations

ADN de marque

Restaurant libanais moderne à Lyon 9 (Vaise). Sain, halal, généreux, rapide. Le design reflète ces valeurs à chaque endroit du site.

Positionnement

Restaurant libanais moderne, sain, halal. Lyon 9 / Vaise. Bowls, wraps et mezzé box à emporter ou en traiteur.

Personnalité

Chaleureux (cream / gold), sain (greens), premium (noir profond + Playfair italique sur les accents).

Ton

Français direct, chaleureux, sans jargon. Orienté action : "Commander", "Composer mon bowl", "Réserver".

03 · Typographie

Deux fontes, pas une de plus

Poppins pour tout le corps et les titres sans emphase. Playfair Display en italique uniquement pour les mots accentués (balise <em>).

05 · Règles

Do & Don't

Les règles simples à respecter dans chaque page.

✓ Do

Utiliser var(--green-d) pour les titres H2.

✗ Don't

Écrire color:#0d4f3d en dur dans style="...".

✓ Do

Playfair Display uniquement en italique sur des <em>.

✗ Don't

Utiliser Playfair Display en régulier comme fonte display.

✓ Do

Charger Poppins + Playfair via l'unique import officiel.

✗ Don't

Importer DM Sans, DM Serif, Instrument Serif, Georgia, ou autre.

✓ Do

CTA "Commandez" toujours en var(--gold) sur tout le site.

✗ Don't

CTA en bleu (--blue est réservé aux liens externes).

✓ Do

Définir :root{--*} une seule fois dans style.css.

✗ Don't

Redupliquer :root{--*} dans chaque bloc <style> local.

✓ Do

Catégorie "Halal" → badge peach + texte green-d.

✗ Don't

Catégorie "Halal" → badge violet #6b21a8 (hors palette).

06 · Checklist

Avant de déployer une nouvelle page

Cocher ces 10 points garantit qu'elle est alignée avec le design system.

☐ Import de fontes identique au template officiel
☐ <link rel="stylesheet" href="style.css"> unique
☐ Aucune redéfinition de :root{--*}
☐ Aucun hex dans les style="..." inline
☐ Aucune mention de DM Sans / Instrument / Georgia
☐ H1 : Poppins 900, <em> Playfair italic
☐ Nav avec variante .nav--light si page intérieure
☐ Breadcrumb présent sous le hero
☐ Meta theme-color="#0d4f3d"
☐ GA4 (ga4.js) chargé
